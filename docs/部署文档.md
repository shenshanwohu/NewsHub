第一步：打开 Vercel

浏览器打开 https://vercel.com/new

第二步：导入仓库

1. 点 Continue with GitHub 登录
2. 找到 shenshanwohu/NewsHub
3. 点 Import

第三步：配环境变量

往下翻到 Environment Variables 部分，点 Add，逐个添加这 5 个：

┌───────────────────────────────┬──────────────────────────────────────────┐
│ 变量名 │ 值 │
├───────────────────────────────┼──────────────────────────────────────────┤
│ NEXT_PUBLIC_SUPABASE_URL │ https://xomafvdipvywvpgtmzqp.supabase.co │
├───────────────────────────────┼──────────────────────────────────────────┤
│ NEXT_PUBLIC_SUPABASE_ANON_KEY │ 从 .env.local 里复制 │
├───────────────────────────────┼──────────────────────────────────────────┤
│ SUPABASE_SERVICE_ROLE_KEY │ 从 .env.local 里复制 │
├───────────────────────────────┼──────────────────────────────────────────┤
│ DATABASE_URL │ 从 .env.local 里复制 │
├───────────────────────────────┼──────────────────────────────────────────┤
│ NEXT_PUBLIC_SITE_URL │ 先填 https://news-hub.vercel.app │
└───────────────────────────────┴──────────────────────────────────────────┘

第四步：部署

点 Deploy，等 1-2 分钟看到 Congratulations! 页面就成功了。

第五步：部署后

拿到 Vercel 域名后，去
https://supabase.com/dashboard/project/xomafvdipvywvpgtmzqp/auth/settings，把 Site URL
改成 Vercel 的域名（不然登录功能会出问题）。
