# NewsHub CMS — Phase 2 开发 Claude 指令集

**前置说明**：

- 已完成 Phase 1（基础设施、Auth、Middleware）
- Supabase 数据库和类型生成脚本就绪
- Next.js + TypeScript + TailwindCSS 环境就绪

---

## 1️⃣ 访客端核心功能

### 任务 2.1：首页新闻列表（分页 + 置顶区域）

```text
目标：生成 /page.tsx 首页
功能：
- 显示置顶新闻在顶部区域
- 列表分页显示普通新闻
- Skeleton Loading 效果
- 每条新闻显示标题、摘要、封面图、发布时间、分类标签
- 点击标题跳转 /news/[slug]
数据源：
- Supabase news 表 + news_categories 多分类
- 按置顶 first, 然后按发布时间倒序
交付物：
- /page.tsx
- FeaturedNews 组件（置顶新闻）
- NewsCard 组件（列表新闻）
任务 2.2：新闻详情页（富文本 + 多分类 + 浏览数）
目标：生成 /news/[slug]/page.tsx
功能：
- 显示新闻标题、封面图、发布时间、作者、分类标签
- 渲染富文本正文（Tiptap 或基础 HTML 渲染）
- 浏览数显示并异步写入 Server Action（view-counter.ts）
- SEO meta + OGP（title, description, image）
交付物：
- /news/[slug]/page.tsx
- CategoryBadge 组件
- RichTextRenderer 组件
- view-counter Server Action
任务 2.3：分类导航 + 分类列表页
目标：
- CategoryNav 组件：顶部显示所有分类，点击跳转 /category/[slug]
- /category/[slug]/page.tsx：显示该分类下新闻列表（含置顶 + 分页）
数据源：
- news_categories JOIN news
交付物：
- CategoryNav.tsx
- /category/[slug]/page.tsx
任务 2.4：全文搜索页
目标：/search/page.tsx
功能：
- 根据关键字 ILIKE 查询 news 表
- 置顶新闻排在前面
- 显示标题、摘要、分类标签、发布时间
- 支持分页
交付物：
- /search/page.tsx
- SearchResults 组件
2️⃣ 管理端新闻管理
任务 2.8：管理后台布局 + 侧边栏导航
目标：/admin/(layout.tsx)
功能：
- 已有 AuthProvider 注入
- 显示侧边栏：新闻管理、系统设置（仅超级管理员）
- 顶部显示 UserMenu（头像缩写 + 用户名 + 登出）
交付物：
- (admin)/layout.tsx
任务 2.9：后台登录页
目标：/admin/login/page.tsx
功能：
- 登录表单（邮箱 + 密码）
- 登录成功跳转 /admin/dashboard 或 redirect 参数页面
- 登录失败显示错误提示
交付物：
- /admin/login/page.tsx
- LoginForm 组件
任务 2.10：后台 Dashboard
目标：/admin/page.tsx
功能：
- 简洁概览页面
- 显示新闻总数、分类总数、近期访问量（可选）
交付物：
- /admin/page.tsx
任务 2.11：新闻列表管理
目标：NewsTable 组件
功能：
- 表格显示新闻：标题、分类、发布时间、置顶状态、操作按钮（编辑/发布/下架）
- 支持筛选（分类、发布时间）、排序（发布时间、置顶）
- 支持置顶开关切换
交付物：
- NewsTable.tsx
- FeaturedToggle 组件
任务 2.12：创建新闻表单（基础版）
目标：NewsForm 组件
功能：
- 输入标题、摘要、封面图
- 多分类选择器（CategoryMultiSelect）
- SEO 字段（SEOFields：title, description, keywords）
- 置顶开关
- 提交时调用 Server Action createNews
交付物：
- NewsForm.tsx
- CategoryMultiSelect.tsx
- SEOFields.tsx
任务 2.13：富文本编辑器集成
目标：RichTextEditor 组件
功能：
- 集成 Tiptap 富文本编辑器
- 支持基本文本格式、列表、链接、图片占位
- 与 NewsForm 表单集成
交付物：
- RichTextEditor.tsx
任务 2.14：编辑新闻 + 保存草稿
目标：/admin/news/[id]/edit/page.tsx
功能：
- 加载已有新闻数据
- 编辑新闻字段、分类、SEO、富文本
- 保存草稿 Server Action
- 发布/下架按钮
交付物：
- /admin/news/[id]/edit/page.tsx
- Server Action saveDraft
- Server Action publishNews / unpublishNews
```
